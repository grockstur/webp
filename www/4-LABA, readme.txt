4 лаба - работа с базой данных
БД создаётся в одном из модулей денвера - phpmyadmin, который доступен по адресу http://localhost/Tools/phpMyAdmin/index.php (при запущенном денвере)
!ВАЖНО локал-адрес БД: 127.0.0.1 (ставится по умолчанию)
	имя пользователя: root
	пароль: mysql
В phpMyAdmin создаём новую бд под названием rivers
в ней создаём таблицу под названием river
с полями:
	id, тип данных INT, индекс PRIMARY, A_I(поставить галочку)
	name, VARCHAR, длина=40
	length, INT
	flowsTo, VARCHAR, 40
	volume, INT
	area, INT

UPD: Некоторые функции могут лагать, т.к. в основном зависят от скорости работы БД. Если не произошло добавление, изменение или удаление записи, обнови страницу
